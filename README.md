# SimStudio

SimStudio je simulačné prostredie určené na vizualizáciu a experimentovanie s formálnymi výpočtovými modelmi. Projekt pôvodne vznikol v roku 2009 ako riešenie vychádzajúce z bakalárskej práce.
Tento repozitár obsahuje aktualizovanú verziu projektu, ktorej cieľom je modernizácia technologického základu, zlepšenie používateľského rozhrania, príprava systému na ďalší rozvoj a rozširenie o gemifikáčné prvky.

## Funkcionalita
SimStudio umožňuje:
- vytváranie výpočtových modelov prostredníctvom grafického rozhrania,
- definovanie stavov a prechodových funkcií,
- simuláciu výpočtu krokovým aj automatickým spôsobom,
- sledovanie priebehu výpočtu v čase.

Podporované modely:
- konečný automat (FA),
- zásobníkový automat (PDA),
- Turingov stroj (TM),
- Abacus machine,
- Random access machine.

## Spustenie
Projekt je určený pre operačný systém Windows.

1. Naklonuj repozitár:
   git clone https://github.com/Darkom-svg/SimStudio_FEI
2. Otvor riešenie v kompatibilnom prostredí (napr. Rider alebo Visual Studio).
3. Spusti projekt ako štandardnú .NET aplikáciu.

## Stav projektu
Projekt predstavuje modernizovanú verziu pôvodného riešenia. Aktuálne prebiehajú úpravy zamerané najmä na:
- aktualizáciu technológie,
- úpravu používateľského rozhrania,
- refaktoring zdrojového kódu,
- rozšírenie funkcionality o gemifikačné prvky.
